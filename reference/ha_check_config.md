# Get Home Assistant configuration validity

Get Home Assistant configuration validity

## Usage

``` r
ha_check_config(ip = NULL)
```

## Arguments

- ip:

  IP address. Default is `NULL`, which uses the local host.

## Value

A list containing the Home Assistant configuration check result.

## Examples

``` r
if (FALSE) { # ha_has_key()
ha_check_config()
}
```
