# Retrieve Home Assistant calendars

Retrieve Home Assistant calendars

## Usage

``` r
ha_get_calendars(ip = NULL)
```

## Arguments

- ip:

  IP address. Default is `NULL`, which uses the local host.

## Value

A list containing Home Assistant calendar information.

## Examples

``` r
if (FALSE) { # ha_has_key()
ha_get_calendars()
}
```
