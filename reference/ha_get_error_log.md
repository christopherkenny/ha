# Retrieve Home Assistant Error Log

Retrieve Home Assistant Error Log

## Usage

``` r
ha_get_error_log(ip = NULL)
```

## Arguments

- ip:

  IP address. Default is `NULL`, which uses the local host.

## Value

A character vector containing the Home Assistant error log.

## Examples

``` r
if (FALSE) { # ha_has_key()
ha_get_error_log()
}
```
